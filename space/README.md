# player-value-ml

LightGBM forecaster for football player market values (6-month horizon),
trained on Kaggle Transfermarkt (~507k valuations). Portfolio piece by
Daniel Regalado. See the [GitHub repo](https://github.com/DanielRegaladoUMiami/player-value-ml)
for training code and feature engineering.
